# POOIplusoRuiDurao
Repositório de exemplos e exercícios de Programação Orientada a Objectos do IPLuso

Para publicar o resultado dos vossos trabalhos, devem criar um novo branch, tendo a seguinte estrutura como referência:
[PRIMEIRO NOME]_[SEGUNDO NOME]_Exercicio_#[SEGUIDO DO NÚMERO DO EXERCÍCIO]

Exemplo:
Rui_Durao_Exercicio_#1
Rui_Durao_Exercicio_#2
